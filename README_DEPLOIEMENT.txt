OUTIL DE RECHERCHE PSM
=======================

Contenu du dossier
------------------
- index.html : page web autonome, avec les donnees integrees.

Deploiement simple
------------------
1. Copier le dossier psm-recherche-deploy sur l'emplacement choisi.
2. Ouvrir index.html depuis un navigateur, ou publier le dossier sur un serveur web/intranet.
3. Si le dossier est publie sur un serveur, l'adresse a communiquer sera par exemple :
   https://votre-serveur/psm-recherche-deploy/

Exemples d'emplacements possibles
---------------------------------
- Serveur intranet interne
- SharePoint ou Teams, si l'ouverture de fichiers HTML est autorisee
- Repertoire reseau partage
- Petit serveur web local ou IIS

Mise a jour des donnees
-----------------------
La page contient les donnees integrees au moment de la generation.
Quand le fichier Excel source change, il faut regenerer une nouvelle version de index.html.

Utilisation smartphone
----------------------
La page est responsive : elle s'adapte automatiquement a l'ecran d'un telephone.
